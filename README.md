# laravel-vue

### Install Lite Server
```shell
sudo npm install -g lite-server
```

### Install Babel
```shell
sudo npm install -g babel-cli
```

### Install Concurrently
```shell
sudo npm install -g concurrently
```

### Install Node Sass
```shell
sudo npm install -g node-sass
```

### Install Gulp Cli
```shell
sudo npm install -g gulp-cli
```

### Install NPM Dependences
```shell
cd vue.js
sudo npm install
```
